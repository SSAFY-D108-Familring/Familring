<h3>📣 기획 배경</h3>

기존의 가족 앱은 주로 일정 관리나 기록에 초점이 맞춰져있습니다.<br>
기존 앱과 달리 **"소통"** 이라는 차별점을 두어 다양한 기능을 통해 가족 구성원들이 서로의 일상을 공유하고 자연스럽게 대화를 이어갈 수 있도록 기획했습니다.<br>
이를 통해 **가족 간의 소통을 촉진하고 유대감을 강화하는 것**을 목표로 합니다.<br>
